How To Import JS Files On HTML Page

Step 1: Create a JS file in 'src/scripts/demo.js'

demo.js:

        export function Hello(uname){
            return `Hello ! ${uname}`;
        }

        export function Addition(a, b){
            return a + b;
        }

        export function Multiplication(a, b) {
            return a * b;
        }

Step 2: Create demo.html

demo.html:

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
            <script type="module">
                import { Hello, Addition } from "./src/scripts/demo.js"; 
                document.querySelector("p").innerHTML = "Addition=" + Addition(10, 20);
            </script>
        </head>
        <body>
            <p></p>
        </body>
        </html>
